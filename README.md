Before attend triple dojo you need to install:
* [Docker](https://docs.docker.com/engine/installation)

* Git

an then run:
````bash
    git clone https://github.com/beeva-manueldepaz/dojo-triple
    chmod a+x ./setup.sh
    ./setup.sh
````
